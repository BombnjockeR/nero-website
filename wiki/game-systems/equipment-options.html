Equipment Options

Custom gear can roll random bonus options (a gacha-style system). Each piece has an Option 1 and Option 2 slot.

Rules

  • Option 1: pay 1,000,000 zeny + 2 Nero Token → random Option 1. Works on gear +0 ~ +10.
  • Option 2: pay 2,000,000 zeny + 4 Nero Token → random Option 2. Works only on gear +7 ~ +10.
  • You can roll an option a maximum of 10 times per equipment. To keep rolling on the same gear, pay 2,000,000 zeny to reset.
  • To keep the option but change only its value, use a Nero Magnifier.
  • Options stack across pieces, so the same stat can appear on multiple slots.
